Question-1:What are the different ways to select an element in the Dom?

There are several ways to select an element in the DOM using JavaScript.document.getElementById() document.getElementsByClassName() document.getElementsByTagName() document.querySelector() document.querySelectorAll() document.getElementById().parentElement document.getElementById().children etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1.innerHTML:Retrieves or sets the HTML content inside an element, including HTML.
2.innerText:Retrieves or sets only the visible text inside an element.
3. textContent:Retrieves or sets all text inside an element, even hidden text.

Question-3:What is event delegation in the DOM?

Event Delegation is a technique in JavaScript where instead of attaching event listeners to multiple child elements, we attach a single event listener to a parent element. The event is then bubbled up from the target child to the parent, where it is handled.

Question-4:What is Event Bubbling in the DOM?

Event Bubbling is a concept in the DOM where an event triggered on a child element bubbles up to its parent, grandparent, and so on, until it reaches the root

Question-5:How do you create, add, and remove elements using JavaScript?

1. Creating an Element:To create a new element, use document.createElement(). This method creates an HTML element.
2. Adding an Element:Once the element is created, you can add it to the DOM using methods like appendChild() or insertBefore().
3. Removing an Element:To remove an element from the DOM, use removeChild().
