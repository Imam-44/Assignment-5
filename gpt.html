What are the different ways to select an element in the DOM?

In JavaScript, elements in the DOM can be selected using `getElementById`, `getElementsByClassName`, `getElementsByTagName`, `querySelector`, and `querySelectorAll`.

What is the difference between `==` and `===` in JavaScript?

The `==` operator compares values with type conversion, while `===` checks both value and type.

How does event delegation work in JavaScript?

Event delegation is a technique where a parent element handles events for its child elements by using event bubbling.
